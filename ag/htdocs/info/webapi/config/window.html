Anatomography map API configuration

Window configuration category

画像サイズ、背景色など、描画画像に関するconfiguration parameter群を設定します。

Configuration parameter

"Window" categoryには以下のパラメータが存在します。

Configuration category デフォルト値 説明 指定可能な値
"Window":{ パラメータの連想配列
"ImageWidth": 500 生成画像の横幅ピクセル数 5000以下の自然数（animationメソッドは1000以下）
"ImageHeight": 500 生成画像の高さピクセル数 5000以下の自然数（animationメソッドは1000以下）
"BackgroundColor": "FFFFFF" 16進数表記の背景色 "000000"～"FFFFFF"
"BackgroundOpacity": 100 背景の不透明度 0～100の整数
"GridFlag": false 画像上に一定間隔の格子を描画するかどうかを指定 true|false
"GridTickInterval": 100 格子の描画単位（mm） 自然数
"GridColor": "FFFFFF" 16進数表記の格子の色 "000000"～"FFFFFF"
}

パラメータの使用例

画像サイズを指定する（"ImageWidth"、"ImageHeight"パラメータ）

生成する画像の大きさは、"Window"カテゴリの"ImageWidth"、"ImageHeight"パラメータで指定します。各々未指定時には500pxに設定されます。

http://lifesciencedb.jp/bp3d/API/image?{"Part":[{"PartID":"FMA5018"}]}


"ImageWidth"、"ImageHeight"にそれぞれ値を指定することで、生成画像のサイズが変わります。なお、幅と高さは同一である必要はありません。
http://lifesciencedb.jp/bp3d/API/image?{"Window":{"ImageWidth":150,"ImageHeight":300},"Part":[{"PartID":"FMA5018"}]}


animationメソッドでもサイズの設定は可能ですが、最大値はimageメソッドで5000pxであるのに対し、animationメソッドでは1000pxに制限されています。
http://lifesciencedb.jp/bp3d/API/animation?{"Window":{"ImageWidth":150,"ImageHeight":300},"Part":[{"PartID":"FMA5018"}]}

背景色を指定する（"BackgroundColor"パラメータ）

画像の背景は、"Window"カテゴリの"BackgroundColor"で色を指定します。

・背景を黒（"000000"）にした場合

http://lifesciencedb.jp/bp3d/API/image?{"Window":{"BackgroundColor":"000000"},"Part":[{"PartID":"FMA5018"}]}

グリッドを表示する（"GridFlag"、"GridTickInterval"、"GridColor"パラメータ）

画像上にグリッドを表示することで、パーツのサイズを把握しやすく描画することができます。
三次元空間上の10mm間隔（"GridTickInterval":10）で、緑色（"GridColor":"00FF00"）のグリッドを描画（"GridFlag":true）するconfigurationは下記のようになります。

http://lifesciencedb.jp/bp3d/API/image?{"Window":{"GridFlag":true,"GridTickInterval":10,"GridColor":"00FF00"},"Part":[{"PartID":"FMA7148"}]}
