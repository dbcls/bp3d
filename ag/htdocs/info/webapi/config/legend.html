Anatomography map API configuration

Legend configuration category

説明文描画有無、説明文など、画像の説明文に関するconfiguration parameter群を設定します。

Configuration parameter

"Legend" categoryには以下のパラメータが存在します。

Configuration category デフォルト値 説明 指定可能な値
"Legend":{ パラメータの連想配列
"DrawLegendFlag": false 画像上に説明文を描画するかどうかを指定します true|false
"LegendPosition": "UL" 説明文の描画位置 "UL"
"LegendColor": "FFFFFF" 16進数表記の説明文描画色 "000000"～"FFFFFF"
"LegendTitle": "" 説明文のタイトル 任意の文字列（URL長の制限範囲内）
"Legend": "" 説明文の内容 任意の文字列（URL長の制限範囲内）
"LegendAuthor": "" 説明文のauthor 任意の文字列（URL長の制限範囲内）
}

パラメータの使用例

説明文を描画する（"DrawLegendFlag"、"LegendColor"、"LegendTitle"、"Legend"、"LegendAuthor"）

説明文を描画するには、"DrawLegendFlag"をtrueにした上で、Title、Legend、Author等のパラメータを設定する必要があります。
