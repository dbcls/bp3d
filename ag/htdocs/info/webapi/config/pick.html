Anatomography map API configuration

Pick configuration category

ピックする最大数、画像上のxy座標など、pick methodに関するconfiguration parameter群を設定します。

Configuration parameter

"Pick" categoryには以下のパラメータが存在します。

Configuration category デフォルト値 説明 指定可能な値
"Pick":{ パラメータの連想配列
"MaxNumberOfPicks": 20 指定数分、パーツ表面を貫通して結果を返します 1 ～ 20の整数
"ScreenPosX": 0 画像上ピック位置のX座標 画像幅ピクセル数内の整数
"ScreenPosY": 0 画像上ピック位置のY座標 画像高さピクセル数内の整数
}
